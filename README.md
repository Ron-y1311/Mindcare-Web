# Mindcare-Web
Plataforma de gestión de salud mental para pacientes, profesionales de la salud y administradores.

---

# 📖 Descripción
MindCare es una plataforma web diseñada para apoyar el bienestar emocional mediante el monitoreo del estado emocional, la gestión de citas psicológicas y herramientas de apoyo para profesionales de la salud.

---

# ✨ Funcionalidades

- Autenticación segura con JWT
- Registro e inicio de sesión
- Gestión de pacientes
- Gestión de profesionales de la salud
- Seguimiento emocional
- Encuestas de evaluación
- Agendamiento de citas
- Historial clínico
- Panel administrativo
